- **LangSmith** A platform where you can deploy, observe and evaluate your agents.

- **Langchain** A high-level framework for building AI agents. Built on top of LangGraph.

- **LangGraph** A low-level framework for building AI agents.

- **Langchain Core** The core components package for LangGraph orchestrations.

- **LangGraph CLI** The official command-line interface for LangGraph, providing tools to create, develop, and deploy LangGraph applications.

- **LangGraph SDK** The Python SDK for interacting with the LangSmith Deployment REST API.

- **DeepAgents** A standalone library for building agents that can tackle complex, multi-step tasks. Built on LangGraph and support CLI.
